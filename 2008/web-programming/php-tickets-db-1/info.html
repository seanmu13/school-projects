My homework is best viewed in firefox 3 or ie 7. It consists of 3 main php files: admin.php, ticket.php, and user.php.

  • admin.php - Main page for admin logins so they can view the tickets, view only open tickets, sort the tickets, view individual tickets, etc.
  • ticket.php - Page that shows the details on a ticket, allows an admin to close/remove/assign tickets.
  • user.php - Allows a user to submit a ticket using their name/email, a subject, and a message

I use style4.css for some basic styling for my tables and fonts.

I use script3.js for client side form checking
